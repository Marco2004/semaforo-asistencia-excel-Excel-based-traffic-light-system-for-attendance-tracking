# Security Policy / Politica de Seguridad

## Espanol

## Version soportada

La version publicada en este repositorio es una demo anonimizada:

- `Sistema_Asistencia_Semaforo_2026_DEMO_ANONIMIZADO.xlsx`

No se da soporte de seguridad para copias privadas, modificaciones internas o versiones que contengan datos reales fuera de este repositorio.

## Privacidad de datos

Este repositorio no debe contener informacion real de empleados, asistencia o datos internos de una organizacion.

Antes de publicar el archivo:

- Los nombres fueron reemplazados por nombres ficticios.
- Los numeros de empleado fueron reemplazados por identificadores ficticios.
- Los supervisores fueron reemplazados por nombres ficticios.
- Los registros se dejaron solo como ejemplo de funcionamiento.

Si adaptas este workbook para uso real, no publiques:

- Nombres reales.
- Numeros de empleado reales.
- Registros reales de entrada y salida.
- Informacion interna de horarios, supervisores o equipos.
- Datos personales, laborales o sensibles.

## Uso seguro

Este archivo no usa macros. Aun asi, si descargas una copia desde internet:

- Abre el archivo solo si confias en la fuente.
- Revisa el contenido antes de habilitar edicion.
- No agregues datos reales si planeas subir el archivo a GitHub.
- Guarda copias internas con controles de acceso si lo adaptas a una empresa real.

## Reporte de problemas

Si encuentras un problema relacionado con privacidad, exposicion de datos o comportamiento riesgoso del archivo:

1. No publiques datos sensibles en un issue publico.
2. Describe el problema sin incluir informacion personal.
3. Abre un issue en GitHub o contacta al mantenedor del repositorio.

## Alcance

Esta politica cubre:

- El archivo demo publicado.
- La documentacion del repositorio.
- Riesgos relacionados con datos ficticios, privacidad y publicacion accidental.

No cubre:

- Archivos privados no incluidos en el repositorio.
- Versiones modificadas por terceros.
- Implementaciones internas con datos reales.

## English

## Supported version

The version published in this repository is an anonymized demo:

- `Sistema_Asistencia_Semaforo_2026_DEMO_ANONIMIZADO.xlsx`

Security support is not provided for private copies, internal modifications, or versions containing real data outside this repository.

## Data privacy

This repository must not contain real employee, attendance, or internal organizational data.

Before publishing the file:

- Names were replaced with fictional names.
- Employee numbers were replaced with fictional identifiers.
- Supervisors were replaced with fictional names.
- Records were kept only as functional examples.

If you adapt this workbook for real use, do not publish:

- Real names.
- Real employee numbers.
- Real check-in and check-out records.
- Internal schedule, supervisor, or team information.
- Personal, labor-related, or sensitive data.

## Safe usage

This file does not use macros. Even so, if you download a copy from the internet:

- Open the file only if you trust the source.
- Review the content before enabling editing.
- Do not add real data if you plan to upload the file to GitHub.
- Store internal copies with access controls if you adapt it for a real company.

## Reporting issues

If you find an issue related to privacy, data exposure, or risky workbook behavior:

1. Do not publish sensitive data in a public issue.
2. Describe the problem without including personal information.
3. Open a GitHub issue or contact the repository maintainer.

## Scope

This policy covers:

- The published demo workbook.
- Repository documentation.
- Risks related to fictional data, privacy, and accidental publication.

It does not cover:

- Private files not included in the repository.
- Versions modified by third parties.
- Internal implementations with real data.
