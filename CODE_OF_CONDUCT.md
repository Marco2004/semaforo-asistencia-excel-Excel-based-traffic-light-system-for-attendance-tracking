# Code of Conduct / Codigo de Conducta

## Espanol

## Comportamiento esperado

- Ser respetuoso y profesional.
- Dar retroalimentacion clara y constructiva.
- Cuidar la privacidad de datos y evitar publicar informacion sensible.
- Colaborar con paciencia cuando se revisen cambios en Excel o documentacion.

## Comportamiento no aceptado

- Comentarios ofensivos, discriminatorios o intimidantes.
- Publicacion de datos personales, laborales o internos.
- Cambios que oculten riesgos de privacidad.
- Spam, ataques personales o discusiones fuera del objetivo del proyecto.

## Aplicacion

Los mantenedores pueden cerrar issues, rechazar pull requests o bloquear participacion si se incumple este codigo.

## Reporte

Si observas una conducta inapropiada, abre un issue sin incluir datos sensibles o contacta al mantenedor del repositorio.

---

## English

## Expected behavior

- Be respectful and professional.
- Provide clear and constructive feedback.
- Protect data privacy and avoid publishing sensitive information.
- Collaborate patiently when reviewing Excel or documentation changes.

## Unacceptable behavior

- Offensive, discriminatory, or intimidating comments.
- Publishing personal, labor-related, or internal data.
- Changes that hide privacy risks.
- Spam, personal attacks, or discussions outside the project scope.

## Enforcement

Maintainers may close issues, reject pull requests, or block participation if this code is violated.

## Reporting

If you observe inappropriate behavior, open an issue without sensitive data or contact the repository maintainer.
